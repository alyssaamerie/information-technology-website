The History of Information Technology

History of Computer Hardware

Generation Time Period Technology Used Characteristics Examples
First Generation 1940s-1950s Vacuum Tubes Large and expensive, slow processing speed. Used machine language (binary) Electronic Numerical Integrator and Computer (ENIAC)
Second Generation 1950s-1960s Transistors Smaller, fatser and more reliable than vacuum tubes. Early programming languages like COBOL and FORTRAN were developed IBM 1401- One of the first commercial computers
Third Generation 1960s-1970s Integrated Circuits (ICs) Even smaller and faster than transistors. Allowed multiple programs to run at the same time (multiprocessing). IBM System/360- First family of computers with compatibilty
Fourth Generation 1970s-present Microprocessors Personal computers (PCs) became common. Faster processing with less power consumption. Graphical User Interface (GUI) introduced IBM PC
Fifth Generation Present-Future Artificial Intelligence (AI) and Quantum Computing Uses artificial intelligence, machine learning and quantum computing. Focuses on automation and cloud computing. Extremely powerful and fast processing AI-powered supercomputers

History of Computer Software

Period Technology Used Characteristics Examples
Early Software (1940s-1950s) Machine Language & Assembly language Programs are written in binary (0s and 1s). Later assembly language made coding easier Machine code for ENIAC, Assembly for IBM 701
High-Level Programming Languages (1950s-1970s) FORTRAN, COBOL, BASIC First high level programmig languages developed. Made coding easier and more human readable FORTRAN (scientifc computing), COBOL (business applications), BASIC (education and early PCs)
Operating Systems & GUI (1970s-1990s) UNIX, DOS Introduction of Operating Systems to manage hardware and software. Graphical User Interface made computers easier to use UNIX (early OS for servers), MS-DOS (text-based OS), Windows 95 (first user-friendly GUI), Mac OS (Apple's GUI OS)
Internet & Web-Based Software (1990s-Present) Web browsers, Cloud computing, Mobile Apps Software moved online with World Wide Web. Development of web browsers (Chrome, Firefox). Cloud-based applications became popular. Mobile apps developed for smartphones Internet Explorer, Google Docs, Microsoft 365, Androids and iOS apps
AI, Cloud computing and Automation (Present-Future) Artifical Intelligence, Cloud Computing Software now integrates AI and automation. Cloud computing allows remote data storage. Advanced cybersecurity and automation tools developed Chatbots (ChatGPT, Siri), AI tools, Quantum Computing Software

History of the Internet and Telecommunications

Telecommunications

The transmission of data, voice and video over distances using Technology
Developments-
  • 1830s- Telegraph invention enabled long-distance communication
  • 1876- Alexander Graham Bell invented the telephone
  • 20th Century- Raido, televison and satellites revoluntionized communication
  • Present- Mobile networks (4G,5G), fiber optics and VolP enhance global connectivity

Internet

A worldwide netwrok connecting computers, allowing for instant communication and data sharing
Developments-
  • 1960s- ARPANET, the first version of the internet, was developed by the U.S military
  • 1980s-1990s- The World Wide Web (WWW) was created by Tim Berners-Lee, leading to web browsers and email services
  • 2000s-Present- High-speed broadband, cloud computing, and social media redefine digital communication
